Create a professional, ATS-friendly resume in minutes. **Easy ATS Resume Builder** is a completely free, highly customizable, and premium UI-based CV maker built with modern web technologies. 

## ✨ Features

- **Live Preview:** See changes reflect instantly on the A4 canvas as you type.
- **ATS-Friendly Classic Layout:** A minimal, highly readable template designed to pass Applicant Tracking Systems (ATS).
- **Download as PDF:** Export your resume instantly in high-quality PDF format.
- **Auto Save Feature:** Save your progress in the browser's local storage and load your data anytime you revisit.
- **Dynamic Formatting:** Built-in intelligent bullet points mapping, GitHub/Portfolio link integrations, and professional data-structure.
- **Premium UI:** Built with Tailwind CSS featuring a sleek, responsive, and beautifully animated user interface.

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS (v4)
- **Icons:** Lucide React
- **Printing/PDF Generation:** React-to-print

## 🚀 Getting Started Locally

Follow these steps to run the project on your local machine:

### 1. Clone the repository
```bash
git clone https://github.com/stackbymazed/resume-builder.git
```

### 2. Install dependencies
```bash
cd resume-builder
npm install
```

### 3. Run the development server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to use the app locally.

## 💡 Why this Resume Builder?
Most online resume tools either hide behind a paywall when you try to download, or inject unwanted watermarks. This platform is open-source, absolutely free, and guarantees a format favored by recruiters.
